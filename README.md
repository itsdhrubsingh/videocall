# Simple WebRTC Video Call

This project provides a minimal sign-in + peer-to-peer video call example using WebRTC with a small Socket.io signaling server.

Run locally:

```powershell
npm install
npm start
# Open http://localhost:3000 in two browser windows and join the same room ID
```
